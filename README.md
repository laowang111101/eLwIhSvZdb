# 学生成绩管理系统 python1293

## 项目概述

学生成绩管理系统是一款基于Python开发的软件，旨在提供学生、教师和管理员多角色之间的便捷切换功能，高效管理学生档案、课程资料和选课流程。

## 技术栈

- 后端框架：Flask
- 数据库：MySQL
- 开发工具：PyCharm
- 数据库管理工具：Navicat

## 功能模块

- 学生档案管理
- 课程资料管理
- 选课流程管理

## 系统角色

- 学生
- 教师
- 管理员

## 运行环境

- Python
- MySQL

## 目录结构

```
项目目录
│
├── app                 # Flask应用目录
│   ├── models          # 数据模型目录
│   ├── views           # 视图层目录
│   ├── templates       # 模板目录
│   └── static          # 静态资源目录
│
├── config              # 配置文件目录
├── database            # 数据库文件目录
└── docs                # 项目文档目录
    ├── design.md       # 设计文档
    └── deployment.md   # 部署文档
```

## 部署步骤

1. 确保Python和MySQL环境已安装。
2. 根据配置文件配置数据库连接。
3. 导入数据库文件到MySQL。
4. 运行Flask应用。

## 核心亮点

- 多角色功能自由切换，满足不同用户的需求。
- 基于 Flask 框架，易于扩展与维护。
- 结合 MySQL 数据库，保证数据存储的稳定性。
- 提供详细设计文档和部署文档，助力学习和项目实践。
